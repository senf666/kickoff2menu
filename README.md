# kick off 2 menu
The Kick Off 2 menu recreated in html 5.

Mobile friendly of course because this isn't 1990 anymore!

For more information about the original game, see [Kick Off 2 on Wikipedia](https://en.wikipedia.org/wiki/Kick_Off_2)

# Usage
Simply open `index.html` in a web browser to view the Kick Off 2 menu.

# Features
* Clean, responsive HTML5 recreation of the classic Kick Off 2 game menu
* Mobile-friendly design with responsive grid layout
* Hover effects on menu items
* Keyboard navigation with arrow keys and space to select

# Keyboard Controls
* **Arrow Keys (↑ ↓ ← →)** - Navigate the menu
* **Space** - Select the highlighted menu item
